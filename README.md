# simplepush
Golang client and library for simplepush.io

# Synopsis
```
simplepush
Usage: simplepush -k key -m message [-t title] [-e event] [-p password]
  -e event
    	Set event
  -k key
    	Set simplepush.io key
  -m message
    	Set message
  -p password
    	Set password, if set send message encrypted
  -t title
    	Set title
```
